## Notes

- Family names are derived from the shared prefix before the trailing variant token (for example `_a`, `_h`, `_void_a`).
- This catalog intentionally focuses on `mus_120_*` music assets only; anonymous fallback IDs such as `mus_1200_*` are not present in the source listfile.
- Runtime integration note: `Midnight_ID_catalog.lua` is loaded before `SoundData.lua`, and the addon uses this catalog as the main source for tracked Midnight music muting.
- Runtime exclusion note: Harandar families are intentionally excluded from active muting through `MIDNIGHT_CATALOG_MUTE_FAMILY_EXCLUSIONS` so Harandar stays native.
- Runtime supplemental note: manual non-catalog `mus_1200_*` seed IDs still live in `BElfVR_NewMusicIDs`.
- Runtime supplemental note: non-Midnight Blizzard tavern / inn / rest-area zonemusic used in supported interiors lives in `BElfVR_SupplementalMusicMuteIDs` and is outside this generated catalog.
